# BracketGrid

A premium, high-contrast bracket racing points tracker, standings manager, and spreadsheet replacer app. Built with **Tauri v2** and **React**, **BracketGrid** is designed to provide race track officials with an ultra-legible, compact, and automated points calculator dashboard.

## Key Features

*   **Custom Class-Based Calculations**: Configure show-up points, normal round wins, and buyback round wins per class.
*   **Worst-Days Drop Logic**: Automatically drops the worst 2 race days of the season for each competitor (requires 3 or more races to activate drops).
*   **Auto-Calculated Standings**: Computes totals, positions, and points gaps to the class leader on the fly.
*   **Compact UI & Large Fonts**: Designed for maximum visibility and ease of use in staging lanes and official booths.
*   **Image Exporter**: Generate high-contrast standings cards (Simple Standings & Full Breakdown Grid) in PNG format for direct sharing on Facebook or forums.
*   **Plain-Text Copy**: Copy tab-delimited text versions of your standings directly to your clipboard for quick forum postings.
*   **Auto-Updates**: The application automatically checks for and downloads updates from GitHub when connected to the internet.
